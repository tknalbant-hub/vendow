# İnsan Odaklı AI Asistanı

Bu proje, yapay zekayı sadece bir araç olmaktan çıkarıp, sohbet edebilen, geçmişi hatırlayan ve insan hızında yanıt veren bir arkadaşa dönüştürmeyi hedefler.

## Özellikler
- 🧠 **Hafıza:** Geçmiş sohbetleri tutar.
- ✍️ **Daktilo Efekti:** İnsan gibi yavaş ve akıcı yanıt verir.
- 🎨 **Modern Arayüz:** Streamlit ile geliştirildi.

## Nasıl Çalıştırılır?
1. `pip install -r requirements.txt`
2. `streamlit run app.py`
